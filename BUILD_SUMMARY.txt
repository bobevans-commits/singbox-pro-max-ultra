### 下一步操作

1. **配置 GitHub Secrets** (在仓库 Settings → Secrets and variables → Actions):
   - `ANDROID_KEYSTORE` (Base64 编码的 keystore 文件)
   - `ANDROID_KEY_PASSWORD`
   - `ANDROID_KEY_ALIAS`

2. **生成 Android 签名密钥**:
   ```bash
   keytool -genkey -v -keystore upload-keystore.jks \
     -keyalg RSA -keysize 2048 -validity 10000 \
     -alias upload -storepass YOUR_PASSWORD -keypass YOUR_PASSWORD
   base64 -w 0 upload-keystore.jks  # 复制输出到 ANDROID_KEYSTORE
   ```

3. **测试构建**:
   - Push 代码触发自动构建
   - 或在 Actions 标签页手动触发 workflow_dispatch

4. **查看详细文档**:
   ```bash
   cat GITHUB_ACTIONS_SETUP.md
   ```
